6. Nuxt API Server Routes and Accepting Payments in Stripe (Anthony Fu)

Goal

What’s an e-commerce app without a way to accept payments? Ummm… NOT an e-commerce app? 😆 That’s right, let’s start collecting payments with Stripe! Along the way you’ll learn about the super cool Nuxt 3 API endpoints.

Prerequisites

  • Starts on branch 7-start
  • Stripe account created
  • See Preparation Guide for more details

Steps

  1. Create a Nuxt server API endpoint and use the Stripe Node SDK to verify the price of the product and then add the product prices to a checkout session
  1. Return the sessions URL from the Nuxt API endpoint and redirect the user to the Stripe checkout page
  1. On successful checkout, stripe should redirect to /checkout/success.

Bonus #1

  1. On the success page clear the users cart of the items they just bought but only after they’ve loaded so you can show an order summary

Helpful Resources

  1. Nuxt 3 Docs: Server Routes
  1. Stripe Node SDK
